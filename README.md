# DTDW

基于 [uni-app 官方 Vue3 + Vite 模板](https://uniapp.dcloud.net.cn/quickstart-cli.html) 创建的基础项目。

## 技术栈

- Vue 3
- uni-app 3.x
- Vite 5

## 常用命令

```bash
# 安装依赖
npm install

# H5 开发
npm run dev:h5

# 微信小程序开发
npm run dev:mp-weixin

# H5 生产构建
npm run build:h5

# 微信小程序生产构建
npm run build:mp-weixin
```

## 目录说明

```
src/
  App.vue          # 应用入口组件
  main.js          # 入口文件
  pages.json       # 页面路由配置
  manifest.json    # 应用配置
  pages/index/     # 首页
```
